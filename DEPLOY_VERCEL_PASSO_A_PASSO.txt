SIMMples POCUS + Trauma — Deploy na Vercel

1) Crie uma conta ou entre na Vercel.
2) Crie um repositório no GitHub chamado simmples-pocus-trauma.
3) Dentro do repositório, clique em Add file > Upload files.
4) Suba TODOS os arquivos desta pasta, incluindo:
   - src
   - public, se existir
   - package.json
   - package-lock.json
   - index.html
   - vercel.json
5) Clique em Commit changes.
6) Na Vercel, clique em Add New > Project.
7) Selecione o repositório simmples-pocus-trauma.
8) Confirme as configurações:
   Framework Preset: Vite
   Build Command: npm run build
   Output Directory: dist
   Install Command: npm install
9) Clique em Deploy.
10) Ao terminar, teste:
   /pocus
   /trauma

Observação: o arquivo vercel.json já foi incluído para evitar erro 404 ao abrir /pocus ou /trauma diretamente.


CONFIGURAR SIMM AI NA VERCEL
----------------------------
1. Abra o projeto na Vercel.
2. Vá em Settings > Environment Variables.
3. Adicione:
   OPENAI_API_KEY = sua chave da OpenAI
4. Opcional:
   OPENAI_MODEL = gpt-4.1-mini
5. Salve e faça novo Deploy.

Sem OPENAI_API_KEY, a suite continua funcionando normalmente, mas o botão SIMM AI mostrará que a IA ainda não está configurada.
